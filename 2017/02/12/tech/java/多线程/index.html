线程

发表于 2017-02-12   |   分类于 技术 ， Java

what

线程 ： 是一个程序内部的顺序控制流

进程 ： 严格意义上讲是一个静态任务

多线程 ： 在同一应用程序中有多个顺序流同时执行

多进程 ： 在操作系统中能同时运行多个任务（程序）

ps：在同一个时间点上，一个CPU只执行一个线程，所谓多线程同时运行，只不过是CPU执行速度较快，多线程的每个程序每个执行一会儿，而产生的错觉给人们以为是多线程同时执行,真正多线程是有双CPU或双核或更多.

多线程的java实现

实现Runnable接口

继承Thread类

实际上，Thread类实现了Runnable接口，当继承了Thread类以后，则不能继承其他类，而Runnable
接口更加灵活，所以原则上，能使用Runnable接口尽量使用Runnable接口.

使用

结束线程的办法
①sleep()时会抛出InterruptException异常，当线程调用interrupt()方法时，线程抛异常，catch里面写”return;”， 结束该线程.
②用逻辑控制，例如”while(flag)”， 当flag为false时结束线程.

ps：极力不推荐使用stop()方法.

join()方法

what：合并到主线程后，将等待该线程执行完毕后，再继续执行后续.

yield()方法

what：暂停当前正在执行的线程对象，并执行其他线程.

setPriority()方法

what：设置线程的优先级.

线程同步

why：当多个线程同时访问一个资源时，有可能造成数据错误.
how：用synchronized 关键字为当前使用对象加锁，仅供当前线程执行完成后，其他线程才可以使用.

死锁

what：一个资源拥有多个对象，当多个线程访问该资源时，占用（锁定）了不同的对象，但该线程需要继续锁定剩余其他对象才能够完成操作时，就会造成死锁.
how：加粗锁的粒度.

面试题

类中一变量b=100,一同步(synchronized)方法中修改b=1000,睡眠5秒，另一普通方法打印b的值，main方法执行同步方法，接着执行普通方法，问：b打印多少？结果1000.

A：synchronized锁定只是当前方法，但其他线程可以访问其他非锁定方法.
