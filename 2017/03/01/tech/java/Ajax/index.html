what

XMLHttpRequest 对象

所有现代浏览器均支持XMLHttpRequest对象（IE5和IE6使用ActiveXObject）XMLHttpRequest 用于在后台与服务器交换数据。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新
