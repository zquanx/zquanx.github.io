常用快捷键

control + H 查看当前类层次结构

control + / 单行注释（英文键盘）

control + shift + / 多行注释（英文键盘）

control + option + o 自动导入所需包，自动去掉多余包

command + shift + F 全局搜索

command + F12 查看当前类所有方法
