关于SimpleDateFormat的yyyy和YYYY的区别

发表于 2018-03-05 更新于 2023-10-25 分类于 Java

问题描述

某系统token生成使用的是YYYY，其他系统调用该系统提供服务时需要校验token，使用的是yyyy，校验token时失败，之前一直没问题，问题发生在2017年12月31日。

问题原因

yyyy 是年，YYYY是 weak year ,表示当前周所属的年份，12月31日是周日，一个礼拜的第一天是周日，所以这一天属于2018年，导致时间不统一，生成的token不一样，调用服务报错

Git相关知识

发表于 2017-05-22 更新于 2023-10-19 分类于 技术

1，下载git，mac有两种方式，一种是homebrew,一种是xcode自带的.

2，创建版本库，也就是仓库，新建一个文件夹，git init 初始化.

3，一般而言，项目所在目录即为工作区，工作区有一个隐藏目录.git，是Git的版本库。版本库中最重要的就是stage（或者叫index）的暂存区，还有Git为我们自动创建的第一个分支master，以及指向master的一个指针HEAD。另外还有日志目录logs.

4，git add 操作，将文件修改添加到暂存区；git commit操作，将暂存区的所有内容提交到当前分支.

5，忽略某些文件时，需要编写.gitignore文件，并且需要提交到版本库，还可以对它做版本管理.

线程

发表于 2017-05-09 更新于 2023-10-19 分类于 技术

what

线程 ： 是一个程序内部的顺序控制流

进程 ： 严格意义上讲是一个静态任务

多线程 ： 在同一应用程序中有多个顺序流同时执行

多进程 ： 在操作系统中能同时运行多个任务（程序）

ps：在同一个时间点上，一个CPU只执行一个线程，所谓多线程同时运行，只不过是CPU执行速度较快，多线程的每个程序每个执行一会儿，而产生的错觉给人们以为是多线程同时执行,真正多线程是有双CPU或双核或更多.

多线程的java实现

实现Runnable接口

继承Thread类

实际上，Thread类实现了Runnable接口，当继承了Thread类以后，则不能继承其他类，而Runnable
接口更加灵活，所以原则上，能使用Runnable接口尽量使用Runnable接口.

使用

结束线程的办法
①sleep()时会抛出InterruptException异常，当线程调用interrupt()方法时，线程抛异常，catch里面写”return;”， 结束该线程.
②用逻辑控制，例如”while(flag)”， 当flag为false时结束线程.

ps：极力不推荐使用stop()方法.

join()方法

what：合并到主线程后，将等待该线程执行完毕后，再继续执行后续.

yield()方法

what：暂停当前正在执行的线程对象，并执行其他线程.

setPriority()方法

what：设置线程的优先级.

线程同步

why：当多个线程同时访问一个资源时，有可能造成数据错误.
how：用synchronized 关键字为当前使用对象加锁，仅供当前线程执行完成后，其他线程才可以使用.

死锁

what：一个资源拥有多个对象，当多个线程访问该资源时，占用（锁定）了不同的对象，但该线程需要继续锁定剩余其他对象才能够完成操作时，就会造成死锁.
how：加粗锁的粒度.

面试题

类中一变量b=100,一同步(synchronized)方法中修改b=1000,睡眠5秒，另一普通方法打印b的值，main方法执行同步方法，接着执行普通方法，问：b打印多少？结果1000.

A：synchronized锁定只是当前方法，但其他线程可以访问其他非锁定方法.
